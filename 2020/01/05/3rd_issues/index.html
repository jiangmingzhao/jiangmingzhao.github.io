  • Local search

Install hexo-generator-searchdb plugin under the folder jiangmingzhao.github.io/
$ npm install hexo-generator-searchdb –save
Add the following content into the Hexo configuration file

Local search

search:
path: search.xml
field: post
format: html
limit: 10000

Set the “enable” to be “true” in the theme configuration file, under “Local search”
