King of Rock and Roll

Posted on 2020-01-08

Elvis Aron Presley (January 8, 1935 – August 16, 1977) was an American singer and actor. Regarded as one of the most significant cultural icons of the 20th century, he is often referred to as the “King of Rock and Roll” or simply “the King”. His energized interpretations of songs and sexually provocative performance style, combined with a singularly potent mix of influences across color lines during a transformative era in race relations, made him enormously popular—and controversial. Years of prescription drug abuse severely compromised his health, and he died suddenly in 1977 at his Graceland estate at the age of 42.

Presley is the best-selling solo artist in the history of recorded music. He was commercially successful in many genres, including pop, country, blues, and gospel. He won three competitive Grammys, received the Grammy Lifetime Achievement Award at age 36, and has been inducted into multiple music halls of fame.

Christmas for Eastern Christian

Posted on 2020-01-07 Edited on 2020-01-08

Jesus’s birth was fixed to be December 25th. Most Christians celebrate on December 25 in the Gregorian calendar. However, part of the Eastern Christian Churches celebrate Christmas on December 25 of the older Julian calendar, which currently corresponds to January 7 in the Gregorian calendar.

How to build a personal website through Github

Posted on 2020-01-05 Edited on 2020-01-08

https://www.jianshu.com/p/4eaddcbe4d12

I’d like to show two different ways to build your personal website through Github.

  • GitHub Pages

  • GitHub account (username)
    The name of the website should be the same as your Github username

  • GitHub repository

  • Git, Homebrew, nodejs, nvm, hexo
    There are a couple of other options to develop the website, such as Jekyll and Hugo, both very popular. See https://www.techiediaries.com/jekyll-hugo-hexo/ for a comparison
    Install HomeBrew and Git
    $ /usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    $ brew install git

Install nvm and Node.js
$ curl https://raw.github.com/creationix/nvm/master/install.sh | sh
$ nvm install 4

install Hexo
$ sudo npm install hexo-cli -g

  • Create website, choose theme, change configuration, write post, test, publish
    $ hexo init username.github.io
    $ cd username.github.io
    $ git clone https://github.com/iissnan/hexo-theme-next themes/next
    The theme “Next” has some issue about the Home menu. It would be better to use another theme.

Configuration

Test
$ hexo s

Install hexo-deployer-git
$ npm install hexo-deployer-git –save

  • Submit the local changes (theme configuration does not need)
    $ hexo clean && hexo g && hexo d

  • Use your own domain instead of the one provided by github
    Just add your domain in the configuration file
